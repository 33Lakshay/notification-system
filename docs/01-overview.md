# Event-Driven Notification System

## Problem

Modern applications need scalable and decoupled notification systems to handle events like user registration, order placement, etc.

## Solution

This project implements an event-driven notification system using queues to decouple producers and consumers.

## Key Features

* Event-based architecture
* Asynchronous processing
* Scalable queue system
* Pluggable notification channels (Email, SMS, Push)

## Tech Stack

* NestJS
* BullMQ + Redis
* PostgreSQL
